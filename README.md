# Employee-Analytics-Dashboard-Power-BI
This project presents an interactive HR analytics dashboards developed using Power BI to analyze employee performance, attrition trends, and key HR metrics. The goal is to provide actionable insights that support data-driven workforce management and strategic decision-making.

## Project Overview
This project focuses on analyzing employee data using Power BI to provide insights into workforce distribution, attrition, and salary trends. The dashboard supports HR teams in making data-driven decisions.

## Key Objective
- Analyze employee distribution across departments and roles
- Monitor attrition and active workforce
- Evaluate salary trends and hiring patterns
- Improve workforce planning and HR decision-making

## Key Performance Indicators (KPIs)
- Total Employees: 1000
- Active Employees: 915
- Employees Left: 85
- Total Salary: 119.14M
- Total Cities: 13

## Dashboard Features
- Employee Distribution by Age Group
- Department-wise Salary Analysis
- Hiring Trends Over Time
- Attrition Analysis
- Job Role Distribution
- Interactive Filters (Department, Gender, Hire Date)

## Tools Used
- Power BI
- DAX
- Power Query
- Microsoft Excel
